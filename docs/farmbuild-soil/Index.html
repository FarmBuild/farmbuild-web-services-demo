Farm Build Soil Area Service

Introduction

  • The Farm Build Soil Area Service allows you to take the property and paddocks features as an input and provides a breakdown by soil class and type into area (Ha) and percentage of property and paddocks.
  • The service is built in REST principles that ensures the URLs conforms to HTTP rules, so a wide range of clients can be used to interact with the service. The Farm Build Soil Area Service requires an input of the Farm Data based on an GeoServer output in Json format as below and a Http POST service.

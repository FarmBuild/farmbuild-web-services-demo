Farm Build WFS Services

Introduction

  • The Farm Build WFS Services published from Geoserver for use in the mapping component and also for general client use. It has two services, soils and parcels.
  • Key layer is soils (ie 1:10,000 mapping in irrigation areas and Gippsland).
  • Property will be published as a convenience but probably a rural filtered set and also updated only annually.
  • The service is built in REST principles that ensures the URLs conforms to HTTP rules, so a wide range of clients can be used to interact with the service. The Farm Build WFS Services is Http Get services.
